## Activity Type Categories

Activities are categorized by type to help users find activities that match their needs:

- **calm** - Relaxing, mindfulness-focused activities
- **creative** - Artistic and expressive activities
- **leisure** - Recreational activities
- **mental** - Cognitive and reflective activities
- **physical** - Movement and exercise activities
- **productive** - Task-oriented activities
- **rest** - Recovery and rest activities
- **social** - Connection and community activities

## Tag Categories

Activities are tagged for filtering based on context:

- **indoor** - Can be done indoors
- **outdoor** - Requires going outside
- **social_call** - Phone/video call activity (filtered at night)
- **daytime** - Best done during daylight hours (filtered at night)

## Combination Behavior

When users select multiple root causes, the system:
1. Combines all alternatives from selected causes
2. Removes duplicates (same activity ID)
3. Filters out already-saved alternatives in "Discover" tab
4. Applies context-aware filtering (weather, time of day)
5. Sorts by friend-specific activities first, then by popularity (likes)

## Notes

- **Popularity (likes):** Used for sorting in the "Discover" tab
- **Friend-specific activities:** Activities like "Call Thomas" are dynamically generated based on user's friends list
- **Custom alternatives:** Users can create their own alternatives, which are stored separately in `customAlternatives` state
- **AI-generated alternatives:** The "AI For You" tab generates contextual alternatives based on selected causes, location, time, weather, and user preferences
